# Bitcoin Jungle

A grass-roots **Bitcoin community + Lightning wallet** in the south of
Costa Rica. We help merchants accept Bitcoin, run Lightning infrastructure,
and contribute everything we build back upstream.

- Open source: [github.com/BitcoinJungle](https://github.com/BitcoinJungle)
- Website: [bitcoinjungle.app](https://bitcoinjungle.app)

The `demo` here makes a real GraphQL request to the Bitcoin Jungle
backend — the same one that powers our wallets — and streams the live
BTC price into your terminal.

Run it: `run projects/bitcoin-jungle/demo`
