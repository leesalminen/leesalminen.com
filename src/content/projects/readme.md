# Projects

Pick one and `cd` into it — each has a `readme.md` and a runnable `demo`.

- **bitcoin-jungle** — A Bitcoin community + Lightning wallet in Costa Rica.
- **lightning-invoice** — A real Lightning tip jar. Generates a payable invoice via my LNURL.
- **nostr-feed** — Streams my latest notes from a public Nostr relay.
- **sql-playground** — Run SQL against a tiny dataset, in your browser, via DuckDB-wasm.

Try `run projects/bitcoin-jungle/demo` from anywhere.
